# 🕒 Login Hours Tracker

A .NET Core MVC application that calculates:

- Total Login Time
- Total Break Time
- Estimated Logout Time

## 🛠 Built With:
- ASP.NET Core MVC
- Razor Views
- Bootstrap (optional)

## ▶️ How to Run
1. Clone the repo
2. Open in Visual Studio
3. Click Run
